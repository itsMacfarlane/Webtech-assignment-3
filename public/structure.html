Structure of a HTML document

Underneath a general overview of the structure of a HTML document is shown. By clicking on the <tags> more information will be provided.

<html>
<head>
<title>
<meta>
<link>
<body>
<header>
<nav>
<article>
<section>
<aside>
<p>
<footer>

<html>

The root level tag of an HTML document. All other HTML tags go inside the HTML tag.

<head>

The head section. Used mostly for metadata.

<title>

The title of an HTML document.

<meta>

Used for specifying various meta information about the document. Meta tags are used within the document <head>.

<link>

Used for linking to external resources. Typically seen in <head>referring to an external style sheet. Different from the <a> (anchor) tag.

<body>

Establishes the body of an HTML document.

<header>

Not to be confused with <head>, <header> typically contains introductory content and layout that goes above the body.

<nav>

Used for specifying a navigational region within a document.

<article>

The <article> tag is new in HTML5. This tag can be used to contain blog entries, forum posts, etc.

<section>

A nonspecific means of breaking up content within a webpage into sections.

<aside>

New in HTML5. The <aside> element is for making sidebars that are placed next to the main content.

<p>

Paragraph. Enclose paragraphs in an article within <p> tags.

<footer>

The footer of a document, below the main content.

Sources: W3schools.com & Eastmanreference.com
